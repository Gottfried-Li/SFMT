/**
 * \mainpage 
 *
 * This is SIMD oriented Fast Mersenne Twister pseudorandom number
 * generator(SFMT).
 *
 * This Project provides pesudorandom number generators of various
 * Mersenne Prime Period: from 2<sup>607</sup>-1 to
 * 2<sup>132049</sup>-1.
 *
 * To get SFMT which has the period of 2<sup>607</sup>-1, you have to
 * specify -DMEXP=607 as a compile option. For example, 
 * @verbatim
 gcc -DMEXP=607 -msse2 -c SFMTp-sse2.c 
@endverbatim
 * will make SFMTp which has the period of 2<sup>607</sup>-1 and is
 * optimized for SSE2.
 *
 * - void init_gen_rand() initializes the generator with a 32-bit
 *   integer seed.
 * - void init_by_array() initializes the generator with an array of
 *   32-bit integers as the seeds.
 * - INLINE uint32_t gen_rand32() generates and returns a pseudorandom
 *   32-bit unsigned integer.
 * - INLINE uint32_t gen_rand64() generates and returns a pseudorandom
 *   64-bit unsigned integer.
 * - void fill_array32() fills the user-specified array with 32-bit
 *   psedorandom integers.
 * - void fill_array64() fills the user-specified array with 64-bit
 *   psedorandom integers.
